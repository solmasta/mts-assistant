# MTS Assistant

MTS Assistant is a mobile-first field tool for HVAC technicians working on real jobsites. It brings together AI guidance, safety-first troubleshooting, a document library, and lightweight field utilities into one simple app that can be used from a phone or tablet.

The project was built as a practical proof-of-concept for JLL MTS-style field workflows: help technicians diagnose problems faster, keep safety steps visible, and surface the right reference material without forcing them to bounce between multiple tools.

## What it does

- AI chat for HVAC troubleshooting and job guidance
- Safety-oriented responses with clear warnings and procedural steps
- Document library with searchable technical references
- Equipment logging for sites and units
- Small field utilities such as belt and filter-related helpers
- Mobile-friendly UI with dark/light theme support

## Tech stack

- Frontend: React 18 + Create React App / react-scripts
- Styling: custom CSS in the app shell
- AI integration: Anthropic-compatible proxy flow
- Hosting: GitHub Pages
- Storage: browser local storage for app state and saved data

## Project structure

- src/ — main React application source
- public/ — static assets and manifest files
- build/ — production build output generated by the build step
- .github/workflows/ — GitHub Actions workflow for Pages deployment

## Running locally

Install dependencies:

```bash
npm install
```

Start the development app:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

## Deployment

The app is published to GitHub Pages. The deployment workflow builds the app in CI and then publishes the generated build output.

## Notes

This project is designed as a field-facing prototype rather than a full production SaaS platform. It focuses on a fast, practical experience for technicians who need answers quickly and clearly while working onsite.

## Status

The app is currently set up for local development and GitHub Pages deployment, with the latest workflow updates aimed at making the published site load reliably.
